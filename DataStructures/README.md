**Cases**

- Best case:  Indicated by sigma Ω notation
- Average case
- Worse case: Indicated by O (big oh) notation

**Some common big oh notation**

- Constant: O(1)
- Linear: O(n)
- Logarithmic: O(log n)
- n log n: O(n log n)
- Quadratic: O(n^2)
- Cubic: O(n^3)
- Exponential: O(2^n)
